## Key Features

* Manage Teachers and their departments, Students, Courses and Exams with grades 
* Clean JavaFX user interface
* Full CRUD functionality 
* Connects to a MySQL Database 
* Includes a script to quickly set up the database
* Administrators can manage teachers, departments, students and courses 
* Teachers can manage exams and set grades 
* Role based access with login authentication 
* Securely stores passwords hashed and salted with PBKDF2
* Comes with three default users: admin, teacher and student 

## How To Run

mvn javafx:run
